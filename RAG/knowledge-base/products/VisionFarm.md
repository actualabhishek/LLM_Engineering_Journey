# VisionFarm

## Summary

VisionFarm brings production-grade computer vision to manufacturing, retail, and security teams without requiring in-house ML expertise. The platform ingests camera or image feeds, runs configurable detection models for defects, safety violations, or inventory anomalies, and routes alerts into existing operational workflows. VisionFarm supports both cloud and edge deployment for latency-sensitive environments.

## Features

- **Custom Defect Models**: Train detection models on a customer's own product images with as few as 200 labeled samples per defect class.

- **Edge Deployment**: Run inference on-site with NVIDIA Jetson or industrial edge boxes for sub-100ms latency and offline resilience.

- **Real-Time Alerting**: Route detections to Slack, email, SMS, or existing MES/SCADA systems.

- **Video and Image Support**: Process both live RTSP camera streams and batch image uploads.

- **Audit Trail**: Every detection is logged with a timestamped image for compliance and quality audits.

- **Human-in-the-Loop Review**: Flagged low-confidence detections route to a review queue for continuous model improvement.
