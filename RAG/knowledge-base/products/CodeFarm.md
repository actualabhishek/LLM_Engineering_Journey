# CodeFarm

## Summary

CodeFarm is an AI coding assistant built for enterprise engineering teams that need more than autocomplete. Beyond code generation, CodeFarm reviews pull requests against a customer's own style guide and architectural conventions, writes and updates tests, and answers natural-language questions grounded in a company's own codebase and internal docs.

## Features

- **Codebase-Aware Generation**: Suggestions and completions grounded in the customer's own repositories, internal libraries, and coding conventions.

- **Automated PR Review**: Flags style violations, potential bugs, and missing test coverage on every pull request.

- **Test Generation and Maintenance**: Automatically generates unit tests for new code and updates existing tests when function signatures change.

- **Natural-Language Codebase Q&A**: Ask questions like 'where is the payment retry logic?' and get cited, accurate answers.

- **IDE and CI/CD Integration**: Plugins for VS Code and JetBrains IDEs, plus native GitHub Actions and GitLab CI integration.

- **On-Premise Option**: Deploy CodeFarm entirely within a customer's VPC for organizations with strict source-code security requirements.
